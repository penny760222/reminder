post /reminders

新增提醒

  • Request
  • Response

Body

Media type: application/x-www-form-urlencoded

Type: object

Properties
  • title: required (string)

    標題

  • remindTime: (string)

    提醒時間

    Examples:

    2017-07-30 15:08
    2017-12-30 08:20
  • tags: (array of string)

    標籤名稱陣列

    Examples:

    [
  "每日工作",
  "非常重要"
]
  • category: (string)

    類別名稱

    Examples:

    個人事項

HTTP status code 201

Body

Media type: application/json

Type: json

Content:

{
  "id" : "1",
  "title" : "Clean Room",
  "remindTime" : "2017-07-15 20:00",
  "isCompleted": false,
  "tags" : [
    {
      "id" : "1",
      "name" : "每日工作"
    },
    {
      "id" : "20",
      "name" : "非常重要"
    }
  ],
  "category" : {
    "id" : "1",
    "name" : "個人事項"
  }
}

HTTP status code 400

Body

Media type: application/json

Type: object

Examples:

{
	error_code: "1001",
	error_message: "無效的參數"
}
